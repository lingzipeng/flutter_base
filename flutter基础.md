## 7、StatefulWidget和StatelessWidget的区别

`StatelessWidget` 是一个无状态的 Widget 类型，意味着它的 UI 在创建后就不会再发生变化。

- 如果你的 UI 不需要根据数据变化而更新，或者是静态的展示内容，可以使用 `StatelessWidget`。
- 如果你的 UI 需要根据数据变化或用户交互而更新，需要包含可变状态，就应该使用 `StatefulWidget`。
